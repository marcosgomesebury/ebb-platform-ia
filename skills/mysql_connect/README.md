# Skill: MySQL Connect

Testa conexão com banco MySQL usando variáveis de ambiente.

## Uso
- Configure as variáveis no .env.example
- Instale dependências: `pip install -r requirements.txt`
- Execute: `python main.py`

## Variáveis de ambiente
- EBB_CONCILIACAO_MYSQL_DEV_HOST
- EBB_CONCILIACAO_MYSQL_DEV_USER
- EBB_CONCILIACAO_MYSQL_DEV_PASSWORD
- EBB_CONCILIACAO_MYSQL_DEV_DATABASE

## Exemplo
```
EBB_CONCILIACAO_MYSQL_DEV_HOST=127.0.0.1
EBB_CONCILIACAO_MYSQL_DEV_USER=root
EBB_CONCILIACAO_MYSQL_DEV_PASSWORD=senha
EBB_CONCILIACAO_MYSQL_DEV_DATABASE=meubanco
```
