# Skill: RDP Connect

Automação para conexão remota via RDP.

## Uso
- Configure as variáveis no .env.example
- Instale dependências: `pip install -r requirements.txt`
- Execute: `python main.py`

## Variáveis de ambiente
- RDP_SERVER
- RDP_SERVER_EBB_CONCILIACAO_USER
- RDP_USER
- RDP_PASS
- RDP_DOMAIN

## Exemplo
```
RDP_SERVER=10.0.0.1
RDP_SERVER_EBB_CONCILIACAO_USER=usuario
RDP_PASS=senha
```
