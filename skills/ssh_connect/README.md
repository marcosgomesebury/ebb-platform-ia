# Skill: SSH Connect

Automação para conexão SSH interativa.

## Uso
- Configure as variáveis no .env.example
- Instale dependências: `pip install -r requirements.txt`
- Execute: `python main.py`

## Variáveis de ambiente
- RDP_SERVER_OVERRIDE
- RDP_SERVER_EBB_CONCILIACAO_DB
- RDP_SERVER
- RDP_USER
- RDP_PASS
- RDP_SSH_PORT
- RDP_SSH_TIMEOUT
- RDP_SSH_KEY_PATH

## Exemplo
```
RDP_SERVER=10.0.0.2
RDP_USER=usuario
RDP_PASS=senha
```
